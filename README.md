# ClubeDaLeitura

![Gif de funcionamento](https://imgur.com/OvL594f.gif)

## Descrição

Gustavo sempre adorou compartilhar suas revistas em quadrinhos com seus
colegas, porém, acabava se perdendo na organização de todas as revistas,
caixas e empréstimos que possuía. Para resolver esse problema, ele
decidiu entrar em contato com a Academia do Programador para poder 
administrar melhor sua coleção de revistas em quadrinhos.

## Tecnologias utilizadas

![](https://img.shields.io/badge/.NET-6.0-blue)
![](https://img.shields.io/badge/C%23-9.0-blue)
![](https://img.shields.io/badge/Windows%20Forms-5.0-blue)

## Funcionalidades

- Cadastro, visualização, edição e exclusão de revistas, amigos e caixas.
- Empréstimo e devolução de revistas.

![](https://imgur.com/A3ssK4W.gif)

- Seleção de cores através de uma paleta interativa.

**Clone o repositório**

```
https://github.com/VictorRSilva05/clube-da-leitura
```

**Navegue até a pasta raiz da solução**

```
cd ClubeDaLeitura
```

**Restaure as dependências**

```
dotnet restore
```

**Navegue até a pasta do projeto**

```
cd ClubeDaLeitura.WinFormsApp
```
